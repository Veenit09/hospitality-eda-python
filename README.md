# hospitality-eda-python
Exploratory Data Analysis on Hospitality dataset using Python (Pandas, Matplotlib)

## Objectives
- Understand booking trends
- Identify revenue drivers
- Analyze occupancy & cancellations
- Discover customer behavior patterns

## Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook

## Key Analysis
- Booking trend analysis
- Revenue distribution
- Occupancy analysis
- Cancellation patterns
- City / property performance

## Key Insights
- Identified peak booking periods
- Found drivers of cancellations
- Highlighted high-performing properties
- Discovered revenue seasonality
